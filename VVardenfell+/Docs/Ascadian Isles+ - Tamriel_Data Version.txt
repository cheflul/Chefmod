Ascadian Isles+ Suran plugin
These plugins cover BOTH the fixes from the non TD version of Ascadian Isles+ and new additions of Tamriel_Data assets to the region and towns. The standard version of Ascadian Isles+ does not need to be loaded alongside this plugin, pick one or the other. The stats for the fixes can be found in the main changelog.
